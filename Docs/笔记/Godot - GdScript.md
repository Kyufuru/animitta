- Godot 的脚本语言使用 GdScript
- GdScript 与 Python 语言基本相同
    数据类型: `int, float, bool, String, Dicts, Array`
    动态类型: `var, const`
    静态初始化: `var str: String = 'hello'`
    函数: `func f(x1: int, x2: String) -> bool:`
        `return true`
    控制流: `if-else, while, for, continue/break`
    类: `class`
