绘画流程: (草图-底色-固有-二分)-(线稿-线稿/底色/固有修正)-光影-细节

主题: 人能够变成史莱姆的话, 他能做什么?

头脑风暴

生肖 / 变形 / 黏胶
生肖 -> 星座 / 动物 / 生态环境 / 群落 / 种族 / 民族 / 阶级 / 皮毛 / 皮毛制品
星座 -> 夜空 / 银河 / 光 / 点 / 连线 / 延时摄影 / 流星 / 大气 / 雨
动物 -> 尾巴 / 爪子 / 牙齿 / 花纹 / 飞行 / 攀爬 / 跳跃 / 游泳 / 挖掘 / 断尾
变形 -> 蜕皮 / 结茧 / 卵 / 孵化 / 变色龙 / 伪装 / 拟态 / 迷彩 / 丛林 / 潮湿
黏胶 -> 胶水 / 粘合 / 粘贴 / 黑色 / 浑浊 / 黑暗 / 阴阳 / 无光 / 丝线 / 连接 / 强韧

1. 蜕皮
    角色皮肤表面生成一层胶液, 然后将其剥离
    当角色在受到致命伤害时, 如果能量槽在50%以上,
    则消耗所有能量槽, 并恢复生命值

2. 结茧
    主动释放, 消耗能量槽, 恢复生命值

3. 卵+孵化
    陷阱, 一种由卵/种子孵化的真菌/植物, 卵被破坏时向周围喷射黏胶/孢子
    卵/种子可以粘在各种地方

4. 变色龙+伪装+迷彩
    蛇变形能力, 消耗能量槽, 获得一定时间的伪装能力, 不会被移动生物阻挡
    可以很方便的攻击那些正面无法突破的敌人

5. 丝线+连接
    陷阱, 球状物体, 收到碰撞/攻击时向攻击来源发射丝线, 将攻击者牵引过去
    有良好的绝缘性, 可以用于创造索道/桥梁, 也可以连接受损电路和破碎物体
    当牵引的物体过重时, 丝线自然下落, 可用于撞开墙壁或充当平台/荡绳
    (类似《漫威蜘蛛侠》的主动技能, 但是是反向的)
    (会不会太复杂了?)

6. 浑浊+黑暗+无光
    被黏胶阻挡的路会遇光分解, 被黏胶污染的水流受电分解, 恢复清澈的状态

7. 民族+种族+阶级+生态环境
    生活在不同环境的同一种族外形会朝适应环境的方向改变
    因环境和习性的改变, 同一种族发展出不同的文化信仰
    东城的种族形成社会, 出现了由变形能力决定的社会分化等...

8. 雨+连接+跳跃
    东城是一个由水流构成传输网络的城市

9. 黏胶+点
    变形会使角色获得穿过缝隙, 特别是墙壁和地板上洞的能力

10. 黏胶+挖掘
    特定变形会使角色获得利用地表黏胶移动的能力
    (类似《斯普拉顿3》的对战技能)

11. 黏胶+挖掘+飞行
    玩家可以通过特定变形, 发射两团特殊黏胶, 建立只有玩家能够穿过的传送通道
    (类似《传送门2》的白色黏胶, 给玩家创建了可以放置传送门的表面)

12. 黏胶+飞行+尾巴
    猴变形可以让玩家使用粘液尾巴, 黏住天花板作为钩索, 荡过部分地形

13. 黏胶+雨
    雨水会弱化变形力量, 水流会洗掉变形

14. 孵化+阴阳
    比如说玩家在受到致命后被完全变形
    (与死亡状态相同, 不过可以自由选择复活时机)
    随后消耗生命值复活(等待死亡状态终止或其他方式)

15. 跳跃+攀爬+粘合
    利用翻转平台/墙壁, 设置定时在两种状态下切换的陷阱

16. 大气+飞行
    沙漠地区使用气流和滑行/滑翔设计机制
